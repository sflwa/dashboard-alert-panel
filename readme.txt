A plugin to move WordPress Admin alerts & notices to a sidebar panel

## Description
This plugin will take dashboard admin alerts and notices from various plugins and move them to a sidebar so that they are not as in your face and keeps the admin clean.
There is an admin bar menu to show the alerts

## Instructions
Install & Activate the plugin - that is it.
